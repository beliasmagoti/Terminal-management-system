## 🔥 Core Features

### 🔐 Authentication & Security
- Laravel Sanctum authentication
- Role-based access control (RBAC)
- Multi-tenant architecture (terminal isolation)

### 🏢 Terminal Management
- Terminal registration & management
- User assignment per terminal

### 🛢 Tank Management
- Fuel capacity tracking
- Safe & critical level thresholds
- Real-time volume updates

### 📡 Sensor Integration
- IoT-like sensor simulation
- Sensor-to-tank mapping
- Live reading updates

### 🔄 Fuel Operations
- Fuel transfers between tanks
- Fuel delivery tracking
- Automated stock updates

### 🔔 Smart Alert System
- Low fuel detection
- Critical threshold alerts
- System anomaly detection

### 📊 Audit & Monitoring
- Full system activity logs
- Device token tracking
- Notification history

---

## 🧠 Business Logic Highlights

- Each terminal operates independently (multi-tenant design)
- Tanks are monitored in real-time via sensors
- Fuel transfers automatically adjust stock levels
- Alerts are triggered based on threshold rules
- All operations are logged for traceability
